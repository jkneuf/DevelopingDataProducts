Introduction

This presentation is part of the Developing Data Products Coursera.org course project submission.

It is an ioslides presentation generated with RStudio.

The Shiny application pitched by this presentation is at https://neuhalfen.shinyapps.io/DevelopingDataProducts/.

The Shiny app source code is available at https://github.com/jkneuf/DevelopingDataProducts.

What is the purpose of the Shiny app?

  • This Shiny application allows a user to analyze banks that offer four different financial products: FBB, FSR, SIMO, and TNMCS.

  • You can see the revenue generated by each bank in four selected regions of the country: East, North, South, and West.

  • In addition, you can see how each financial product performs in each of the four regions of the country.
